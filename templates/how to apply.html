How to Apply Online

User Registration

1. User can register on the website for availing services.
2. A soft copy of the documents listed below must be uploaded during registration process(Please note
    that system will not allow uploading of document in PDF format with a maximum size of 5 MB.)
  • A soft copy of your PAN card.
  • all the license/certificate related to unit.
3. On successful registration the user will recive an activation pin code by email and SMS at their registered
     email address and mobile number. This code is used to validate user registration to complete the
      registration process.

Updating Profile, Unit and User Details

1. On successful activation of registration, Update User Profile, Unit and Unit User Details. These details are
     necessary for processing of online applications for the services mentioned above.

Submission of Application
1.  User can apply for the services mentioned above.
2.   For every application submitted the applicant receives an email and SMS intimation.
